# 구현 완료 기능 명세서

본 프론트엔드 프로젝트에 구현된 주요 기능 목록을 리스트업한 문서임.

## 1. 인증 및 세션 관리

*   **임시 인증 로직**
    *   `localStorage`를 활용한 로그인 및 회원가입 UI 구현
    *   비로그인 사용자 접근 차단 및 인증 화면 렌더링
*   **세션(대화방) 관리 시스템**
    *   좌측 사이드바를 통한 신규 세션 생성, 기존 세션 선택, 세션 삭제 기능
    *   단일 세션 내에 '이력서 첨삭 로그'와 '모의 면접 로그' 데이터를 통합 저장하는 상태 관리 구조 설계

## 2. 이력서 첨삭 시스템

*   **파일 업로드 UI**
    *   PDF, TXT 파일 드래그 앤 드롭 및 클릭 업로드 인터페이스 구현
*   **첨삭 결과 비교 뷰어**
    *   첨삭 전 원본 데이터와 첨삭 후 결과물을 좌우 분할 형태로 비교할 수 있는 레이아웃 렌더링
*   **하이라이팅 및 수정 사유 툴팁**
    *   문장 단위의 수정 사항에 하이라이트 CSS 효과 적용
    *   마우스 호버 시 수정 사유를 설명해주는 툴팁 UI 구현

## 3. AI 모의 면접 시스템

*   **채팅 인터페이스**
    *   메시지 송수신 컴포넌트 구현 및 하드코딩된 AI 응답 타이핑 시뮬레이션 적용
    *   이력서 첨삭 데이터 유무에 따른 동적 로딩 텍스트 분기 처리 (이력서 분석 및 답변 생성 중 / 답변 생성 중)
*   **면접 환경 설정**
    *   상단 설정 탭을 통한 면접 스타일 선택 (일반 면접, 압박 면접)
    *   면접관 아바타 및 성별 선택 제어 기능
*   **음성 입력 UI**
    *   마이크 권한 요청 및 답변 녹음 시작/중지 버튼 상태값 관리 (STT API 연동을 위한 사전 준비)

## 4. 면접 종료 및 결과 리포트

*   **면접 종료 로직 분리**
    *   상단 [면접 분석 및 종료] 버튼: 면접 종료 처리 후 백엔드 결과 데이터를 모달창으로 출력
    *   하단 채팅창 [+] 메뉴 버튼: 피드백 분석 과정 없이 즉시 대화만 강제 종료 처리
*   **종료 후 대화 잠금 제어**
    *   면접이 완료된 상태(`isCompleted: true`) 전환 시 채팅 텍스트 입력창, 파일 첨부, 마이크 버튼 영구 비활성화 처리
*   **피드백 리포트 모달 출력**
    *   연산 부하가 큰 점수 시스템 및 차트를 배제하고 텍스트 기반 결과 렌더링
    *   주요 강점, 보완해야 할 점, AI 종합 피드백 등 3개 섹션 분리 및 스타일링 적용

## 5. UI/UX 디자인 시스템

*   **테마 및 스타일링**
    *   다크 테마를 기본으로 한 컴포넌트 색상 일관성 적용
    *   부드러운 상태 전환 애니메이션(CSS Transition/Animation) 및 마이크로 인터랙션 구현
*   **반응형 레이아웃**
    *   모바일/태블릿 화면 대응을 위한 좌측 사이드바 토글 숨김 기능 구현
